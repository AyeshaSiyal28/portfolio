HTML5 & CSS3

This page provides detailed HTML5 and CSS3 notes with examples and best practices for portfolio or interview purposes.

1. HTML5 Basics

HTML5 is the latest standard of HTML, introducing semantic elements, multimedia support, and improved form controls.

  • Doctype declaration: <!DOCTYPE html>
  • Semantic tags: <header>, <footer>, <article>, <section>
  • Multimedia support: <audio>, <video>
  • Form improvements: new input types like email, date, number, range
<!DOCTYPE html>
<html>
  <head>
    <title>HTML5 Example</title>
  </head>
  <body>
    <header><h1>Welcome to HTML5</h1></header>
    <section><p>This is a section</p></section>
    <footer><p>Â© 2025 My Website</p></footer>
  </body>
</html>

2. HTML5 Forms

HTML5 introduces new input types and validation features for creating interactive forms.

<form>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" required>

  <label for="email">Email:</label>
  <input type="email" id="email" name="email">

  <label for="dob">Date of Birth:</label>
  <input type="date" id="dob" name="dob">

  <button type="submit">Submit</button>
</form>

3. HTML5 Multimedia

Embedding audio and video without plugins using HTML5 tags.

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

4. CSS3 Basics

CSS3 introduces new features like rounded corners, shadows, gradients, transitions, animations, and responsive design capabilities.

  • Selectors: class, id, attribute, pseudo-class
  • Box model: margin, border, padding, content
  • Advanced layouts: Flexbox, Grid
  • Transitions & Animations
  • Media queries for responsive design
p {
  color: blue;
  font-size: 18px;
  margin: 10px;
  padding: 5px;
  border: 2px solid black;
  border-radius: 8px;
}

5. CSS3 Flexbox

Flexbox is used for one-dimensional layout alignment and distribution of space among items.

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
<div class="container">
  <div>Item 1</div>
  <div>Item 2</div>
</div>

6. CSS3 Grid

Grid provides two-dimensional layout for precise placement of elements.

.grid-container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  gap: 10px;
}
<div class="grid-container">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>

7. CSS3 Transitions & Animations

Add smooth visual effects to enhance user experience.

.button {
  background-color: blue;
  color: white;
  padding: 10px 20px;
  transition: background 0.5s;
}

.button:hover {
  background-color: darkblue;
}

@keyframes slide {
  from { transform: translateX(0); }
  to { transform: translateX(100px); }
}

.box {
  animation: slide 2s infinite;
}

8. CSS3 Media Queries & Responsive Design

Media queries make web pages responsive across different devices.

@media screen and (max-width: 768px) {
  body {
    background-color: lightgray;
  }
  .container {
    padding: 10px;
  }
}
