JavaScript

This page provides detailed JavaScript notes with examples and best practices for portfolio or interview purposes.

1. JavaScript Basics

JavaScript is a high-level, dynamic programming language used for creating interactive web pages.

  • Variables: var, let, const
  • Data types: string, number, boolean, object, undefined, null
  • Operators: arithmetic, comparison, logical
let name = "Ayesha";
const age = 25;
var isActive = true;

console.log("Name:", name, "Age:", age, "Active:", isActive);

2. Functions & ES6

Functions are reusable blocks of code. ES6 introduces arrow functions, default parameters, template literals, and more.

// Function declaration
function add(a, b) {
  return a + b;
}

// Arrow function
const multiply = (a, b) => a * b;

// Template literals
console.log(`Sum: ${add(2,3)}, Product: ${multiply(2,3)}`);

3. Objects & Arrays

Objects store key-value pairs; arrays store ordered lists of items.

// Object
const user = { name: "Ayesha", age: 25 };
console.log(user.name);

// Array
const colors = ["red", "green", "blue"];
colors.push("yellow");
colors.forEach(color => console.log(color));

4. Loops & Conditionals

Control the flow of code using loops and conditional statements.

for(let i=0; i<5; i++) {
  console.log("Number:", i);
}

if(colors.includes("red")) {
  console.log("Red is in the list");
} else {
  console.log("Red not found");
}

5. DOM Manipulation & Events

JavaScript can dynamically change HTML content and respond to user actions.

const button = document.querySelector("#myBtn");
const text = document.querySelector("#myText");

button.addEventListener("click", () => {
  text.textContent = "Button Clicked!";
});

6. Asynchronous JavaScript

Handle asynchronous operations using callbacks, promises, and async/await.

// Promise
fetch("https://jsonplaceholder.typicode.com/users")
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));

// Async/Await
async function getUsers() {
  try {
    const res = await fetch("https://jsonplaceholder.typicode.com/users");
    const data = await res.json();
    console.log(data);
  } catch(err) {
    console.error(err);
  }
}
getUsers();

7. Local Storage & JSON

Store and retrieve data in the browser and work with JSON data.

// Local Storage
localStorage.setItem("name", "Ayesha");
console.log(localStorage.getItem("name"));

// JSON
const userJSON = '{"name":"Ayesha","age":25}';
const userObj = JSON.parse(userJSON);
console.log(userObj.name);

8. Error Handling

Catch errors gracefully using try-catch blocks.

try {
  let result = riskyOperation();
  console.log(result);
} catch(error) {
  console.error("An error occurred:", error);
}
