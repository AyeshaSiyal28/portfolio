API Integration

This page covers everything about API Integration in front-end and full-stack applications, from basics to advanced usage, with examples.

1. What is an API?

API (Application Programming Interface) is a set of rules that allows different software applications to communicate with each other. APIs are commonly used to fetch or send data between frontend and backend systems.

  • RESTful APIs use HTTP methods like GET, POST, PUT, DELETE.
  • Data is usually exchanged in JSON format.
  • APIs can be public, private, or partner APIs.

2. How to Integrate APIs

API integration involves sending requests to an API endpoint and handling the response in your application.

  // Using fetch API in JavaScript
  fetch('https://jsonplaceholder.typicode.com/users')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Error:', error));

This is a basic GET request that retrieves user data from a public API.

3. Example 1: GET Request

Fetching a list of users from an API using React useEffect:

  import React, { useEffect, useState } from 'react';
  
  function UsersList() {
    const [users, setUsers] = useState([]);
  
    useEffect(() => {
      fetch('https://jsonplaceholder.typicode.com/users')
        .then(res => res.json())
        .then(data => setUsers(data));
    }, []);
  
    return (
      <ul>
        {users.map(user => (<li key={user.id}>{user.name} ({user.email})</li>))}
      </ul>
    );
  }

4. Example 2: POST Request

Sending data to an API endpoint:

  fetch('https://jsonplaceholder.typicode.com/posts', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      title: 'My Post',
      body: 'This is a post content',
      userId: 1
    })
  })
  .then(res => res.json())
  .then(data => console.log(data));

5. Example 3: Async/Await

Using modern async/await syntax for API calls:

  async function getPosts() {
    try {
      const res = await fetch('https://jsonplaceholder.typicode.com/posts');
      const data = await res.json();
      console.log(data);
    } catch(error) {
      console.error('Error:', error);
    }
  }
  getPosts();

6. Example 4: Axios Library

Axios is a popular library for HTTP requests, providing easier syntax and automatic JSON parsing:

  import axios from 'axios';
  
  axios.get('https://jsonplaceholder.typicode.com/users')
    .then(res => console.log(res.data))
    .catch(err => console.error(err));
  
  // POST request with Axios
  axios.post('https://jsonplaceholder.typicode.com/posts', {
    title: 'New Post',
    body: 'Post content',
    userId: 1
  })
  .then(res => console.log(res.data));

7. API Integration Best Practices

  • Always handle errors using try/catch or .catch()
  • Use environment variables for API keys
  • Optimize API calls and avoid unnecessary requests
  • Secure sensitive data when sending or receiving APIs
  • Use async/await for cleaner and readable code
