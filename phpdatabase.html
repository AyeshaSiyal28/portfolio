PHP & MySQL Database

This page provides detailed PHP and MySQL database notes with examples and best practices for portfolio or interview purposes.

1. PHP Basics

PHP is a server-side scripting language used to create dynamic web pages.

<?php
  echo "Hello, World!";
?>

2. Variables & Data Types

  • Variables: start with $
  • Data types: string, integer, float, boolean, array, object, null
<?php
  $name = "Ayesha";
  $age = 25;
  $isActive = true;

  echo "Name: $name, Age: $age";
?>

3. Arrays & Loops

Store multiple values and loop through them.

<?php
  $fruits = ["Apple", "Banana", "Orange"];

  foreach($fruits as $fruit) {
    echo $fruit . "<br>";
  }
?>

4. Functions

Reusable blocks of code.

<?php
  function greet($name) {
    return "Hello, $name!";
  }

  echo greet("Ayesha");
?>

5. Forms & Validation

Process user input from HTML forms safely.

<form method="post" action="process.php">
  <label>Name:</label>
  <input type="text" name="name" required>
  <button type="submit">Submit</button>
</form>

<?php
  if($_SERVER["REQUEST_METHOD"] == "POST"){
    $name = htmlspecialchars($_POST['name']);
    echo "Hello, $name!";
  }
?>

6. MySQL Database Integration

Connect PHP to MySQL database to store and retrieve data.

<?php
  $servername = "localhost";
  $username = "root";
  $password = "";
  $dbname = "mydb";

  $conn = new mysqli($servername, $username, $password, $dbname);

  if($conn->connect_error){
    die("Connection failed: " . $conn->connect_error);
  }
  echo "Connected successfully";
?>

7. CRUD Operations

Create, Read, Update, Delete records in MySQL using PHP.

// Create
<?php
$sql = "INSERT INTO users (name, email) VALUES ('Ayesha','a@example.com')";
$conn->query($sql);
?>

// Read
<?php
$result = $conn->query("SELECT * FROM users");
while($row = $result->fetch_assoc()){
  echo $row['name'] . "<br>";
}
?>

8. Prepared Statements & Security

Prevent SQL injection using prepared statements.

<?php
$stmt = $conn->prepare("INSERT INTO users (name,email) VALUES (?, ?)");
$stmt->bind_param("ss", $name, $email);

$name = "Ayesha";
$email = "a@example.com";
$stmt->execute();
$stmt->close();
$conn->close();
?>
